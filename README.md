# Student Management System (CLI)

Simple Python assignment implementing a command-line Student Management System.

Requirements
- Python 3.x

Run
```
python student_management.py
```

Features
- Add / View / Update / Delete students
- Calculate average marks
- Store data in `students.json`
- Sort students by marks (descending)

Notes
- Data is auto-saved to `students.json` after changes.
- Keep the script simple so it's easy to explain in interviews.
